# Lab 05 Project Context

This folder is part of **Lab 05: AI Agents - Terminal-Based AI Operations** for CISC 395. 

## Project Overview
The folder contains a variety of files for AI-assisted operations, including:
- **`data/student_grades.csv`**: A CSV file containing student names and scores for analysis.
- **`code/`**: Contains two Python files (`buggy_sort.py` and `calculator.py`) used for debugging and code explanation tasks.
- **`papers/`**: Contains a research paper PDF for automated summarization.
- **`messy/`**: A folder with 10 poorly-named files that need to be reorganized and renamed.

## Goals
1. Identify and fix bugs in the Python scripts.
2. Summarize research papers into structured reports.
3. Analyze student grade data and generate performance reports.
4. Use AI to systematically reorganize the "messy" digital workspace.
